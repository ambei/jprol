PROL ENGINE
-----------------

It is a Java Based Embeddable Prolog engine supports the Edinburgh Prolog style. It has the Embedded GUI NotPad.
The First Version of the Engine was developed in 2010 by Igor Maznitsa (http://www.igormaznitsa.com) as a project in bounds of Prolog learning. The Project is not recommended for usage in production.
The Project is licensed under Apache License 2.0

You can start the Embedded NotePad as java -jar jprol-<version>.jar 
If you need more info, just ask the author or see tests and sources.
